prueba tecnica bi pruebas automatizadas

Instalar maven
```bash
sudo apt-get install maven
```

Instalar jdk
```bash
sudo apt search openjdk

sudo apt install openjdk-17-jdk

javac --version

java --version
```

Insalar paquete en VSCode (CRTL+P)
```bash
ext install vscjava.vscode-java-pack
```
